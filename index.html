Chameleon Ultra 本地管理台

基于 Web Serial（USB）或 Web Bluetooth（BLE）直连设备，用于对你自己固件（v2.2.0-DAHAOREN）进行在线设置与调试。单页应用：连接一次即可在各功能页之间切换，连接不会丢失。完全本地运行，无需联网。

开始

点右上角「连接设备」前，可先用顶栏下拉框选择连接方式：USB 串口（数据线连接，浏览器弹出串口列表选 Chameleon Ultra，USB VID 0x6868 / PID 0x8686）或 蓝牙 BLE（与官方 lab 一致，浏览器扫描并选择名为 ChameleonUltra 的设备）。两种均需 Chrome / Edge。

提示：设备需处于「卡片模拟 / 设置」状态（非 DFU）。USB 连接失败可重插拔；蓝牙需在系统蓝牙中先配对/靠近设备。
